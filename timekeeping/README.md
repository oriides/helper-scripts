# Work time management using `klog`

## Requirements

+ [`klog`](https://klog.jotaen.net/)

## Usage

Move the scripts to a location that is on your path (e.g. `/usr/local/bin/`)
for easy access from every terminal.

Start time tracking

```bash
start-the-klog
```

Stop time tracking

```bash
stop-the-klog
```
